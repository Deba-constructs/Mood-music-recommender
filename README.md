# 🎧 Mood-Based Music Recommender

An AI-powered web app that detects user emotions from text and suggests music accordingly.

## 🚀 Features
- Emotion detection using NLP
- Machine Learning model (TF-IDF + Logistic Regression)
- Interactive UI with Streamlit
- Real-time mood-based recommendations

## 🧠 Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas

## ▶️ How to Run

1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py

## 📊 Model
- Trained on emotion dataset
- Supports moods like Happy, Sad, Angry, Bored
